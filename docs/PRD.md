Product Overview
FlowLens 是一款以 AI 驱动的心流执行助理 。本项目当前阶段聚焦于手机与电脑双端协同的 MVP 版本，致力于帮助现代职场人与学生降低自律成本，进入专注状态 。

Core Features


备忘录式动态排期：支持模糊时间块（早/中/晚）与精确时间节点混用 。电脑端展示丰富日历看板，手机端保持极简列表，两端数据实时同步 。


手动心流锁定（专注模式）：提供全天日程总览，用户手动锁定单一任务后进入沉浸式计时界面，视觉上仅保留当前任务状态 。电脑端额外支持全屏或悬浮窗沉浸模式 。


无缝对话复盘：提供便捷的录音入口供用户录入“碎碎念” 。系统后台自动转录文字并提取“今日成就、遇到的问题、情绪状态”，生成结构化报告同步至云端 。用户可在电脑端查看详细的周报与情绪趋势图 。
